Experimenting on website publishing services

Github pages Netlify WordPress

Netlify Pros:

Netlify is a cheap and easy to follow for basic design of website:

  • Easy to set up - Neat user interface - Free Netlify domain - Debugging logs - Custom domains - Github integration
  • Easier to use and helps deploy static website from Github/Gitlab within minutes
  • Netlify functions can help you to automate a lot of things on your website
  • Inbuilt features like forms submissions and analytics


Cons:


Netlify is not the beginner-friendliest platform for a website. User has to know how to code and integrate the features on the website.
The UI to manage builds and custom domains is really straightforward to use, operate and maintain though.

"If you pull the wrong repo in, you can't fix it or remove it. So you end up with multiple sites that are nearly the same."

Netlify offers a free publishing environment and a domain, though there’s a possibility to buy a more comprehensive service as well.

Conclusion is that this service might be suitable for someone who has already a bit of experience on coding and has a multifaceted view on web publishing practises.

  • Seems to be quite popular amongst professional developers
  • Suitable for creating web apps and sites

As a beginner I would rather pick another publishing platform at this moment.
